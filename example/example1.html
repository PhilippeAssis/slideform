.selections

Esse é um modelo de lista para seleção de itens. Você pode subistitulo pelo tradicional select ou radio.
Nesse exemplo usamos data-required como atributo da input, assim, caso não seja selecionado nenhum item, a função required() será chamada.

Required

Assim como beforeNext e beforePrev, você pode criar seu próprio required atribuindo ao slideform. Assim você obterá o retorno de todos os itens requiridos do quadro atual que não forão preenchidos ou marcados.
Você pode ativar o requirimento de um item com o atributo data-required contendo uma mensagem de alerta, exemplo: data-required="Esse item é obrigatório", ou usando somente o atributo required do HTML5, nesse caso você só terá de retorno o elemento em ação. 

var myForm = $("#myForm").slideform()

myForm.required = function(response){
    console.log(response)
}

Antes de enviar nosso formulário, vamos dar uma olhada nesse código

O código a baixo é um exemplo de envio do formulário.
Primeiro ele identifica qual o quadro que vai enviar o fomulário, no nosso caso o "sendForm" depois ele pega todos os valores usando this.result() , envia o formulário (essa é a parte comentada, nesse exemplo ele só exibe os dados enviados no próximo quadro) e passa para o próximo quadro usando this.nextSection() 

var myForm = $("#myForm").slideform();

myForm.beforeNext = function () {
    if (this.current.name == 'sendForm') {
        var result = this.result();
        $('#result').text(result);

        /*
        $.post('submit.php', result, function (data) {
            console.log(data)
        })
        */

        this.nextSection()
    }

    return true;
}
